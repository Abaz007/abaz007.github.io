Sooda Mobile Application

Overview

Sooda is a mobile application for a virtual bank. It enables it’s customers to manage their accounts and carry out transactions effectively without going to the bank physically.

Goal

The main goal of this project was to redesign the user interface for the mobile application. The first designs that were made could not be published because the user interface was not good enough. Hence the reason for the Redesign

user interface design

Onboarding Process

The onboarding process follows through from the splash screen to the login screen. If a user doesn’t have an account, the user can create one before signing in.

Home Screen

Some of the major functionalities are right here on the home screen. Features that helps the user to send and receive money. Other features such as to check transaction history and so on are also here on the home screen.

Cards

The Cards Tab Contains the different cards used for transaction by the user. It also shows the various income flows and expenses of the user.

Payments

This is screen contains the monthly value payment for companies, municipalities or institutions that issues Invoices due to the citizens.

Exchange

This allows the user to make transactions to other banks and also schedule future transactions.

What I learnt from this Project

I learnt that providing a good user interface also goes further to enhance the user experience. Hence the two work in line with each other to make a project successful.
